# kairos-skill

Agent skill for KairOS — calendar, tasks, leads, and budget via the Agent CLI.

| File                                           | Purpose                                      |
| ---------------------------------------------- | -------------------------------------------- |
| [`SKILL.md`](./SKILL.md)                       | AI agents — setup, calling tools, web vs CLI |
| [`references/auth.md`](./references/auth.md)   | Login, flags, credentials, troubleshooting   |
| [`references/tools.md`](./references/tools.md) | Tool names and JSON payloads                 |

**Install:**

```bash
npm install -g kairos-cli
```

Add **kairos-skill** using the `npx skills add …` command from **KairOS Dashboard → Connect your AI agent**.

Full setup: [`SKILL.md`](./SKILL.md).
